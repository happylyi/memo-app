# 个人备忘录App ✨
一个简洁、实用的前端备忘录应用，支持新增、删除、编辑、标记重要等功能，数据本地存储，适配移动端。

## 🎯 项目亮点
- ✅ **完整敏捷流程**：基于 PingCode Scrum 敏捷开发，从需求规划到版本发布全流程落地
- ✅ **核心功能齐全**：新增/删除/编辑/标记重要/本地存储/移动端适配/暗色模式
- ✅ **0基础可运行**：纯 HTML+CSS+JS 实现，双击 `index.html` 即可打开使用
- ✅ **文档完善**：配套 PRD、CHANGELOG、README，适合求职作品集展示

## 🚀 快速开始
1. 下载项目文件
2. 找到 `index.html` 文件
3. 双击打开，即可在浏览器中使用备忘录

## 📋 核心功能
### V1.0.0 基础功能
- **新增备忘录**：输入内容后点击保存，最多支持50字
- **删除备忘录**：点击删除按钮，确认后移除记录
- **标记重要**：点击标题切换红色加粗样式，突出优先级
- **本地存储**：刷新页面数据不丢失，数据仅存在本地
- **移动端适配**：手机端布局自适应，操作流畅

### V1.1.0 新增功能
- **编辑备忘录**：点击编辑按钮回填内容，保存后仅更新原条目

### V1.2.0 新增功能
- **本地存储优化**：数据刷新不丢失、不乱序，可选导出JSON

### V1.3.0 新增功能
- **日间/暗色模式切换**：右上角一键切换，自动记忆用户偏好，移动端优化点击区域
